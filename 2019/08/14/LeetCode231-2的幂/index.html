LeetCode231-2的幂
2019-08-14

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。
示例 1:
输入: 1
输出: true
解释: 20 = 1
示例 2:
输入: 16
输出: true
解释: 24 = 16
示例 3:
输入: 218
输出: false

解题思路：
1.位运算
如果是2的幂，则二进制应该只有最高位上是1，其他都是0；
基于上面的结论，n-1应该除了最高位是0（与n最高位对应的位置），其他都是1；
因此，n&(n-1) 应该等于0。 

class Solution:
def isPowerOfTwo(self, n: int) -> bool:
    if n > 0 and n&(n-1) == 0:
        return True
    else:
        return False

2.迭代法
用此数一直除以2，判断是否整除。

class Solution:
def isPowerOfTwo(self, n: int) -> bool:
    if n<=0: return False
    while n>1:
        if n%2==0:
            n /= 2
        else:
            return False
    return True
