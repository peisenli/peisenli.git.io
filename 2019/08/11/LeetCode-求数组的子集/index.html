LeetCode78.子集

给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。
说明：解集不能包含重复的子集。

示例:
输入: nums = [1,2,3]
输出:
[[3],[1],[2],[1,2,3],[1,3],[2,3],[1,2],[]]

思路：在求取数组子集时，对每个数据元进行遍历后在原有的List上进行追加即可获得元素的子集。
python实现： 

class Solution:
def subsets(self, nums: List[int]) -> List[List[int]]:
    result = [[]]
    nums.sort()
    for num in nums:
            result += [item+[num] for item in result]
    return result
